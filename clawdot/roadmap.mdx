## 近期重点

<AccordionGroup>
  <Accordion title="Agent 控制台" icon="gauge-high">
    Web 管理面板，提供：
    - Agent 创建与密钥管理
    - 用户绑定状态查看
    - 订单记录与统计
    - 用量监控与限流配置
  </Accordion>

  <Accordion title="Agent SDK" icon="toolbox">
    面向开发者的 SDK，简化 Agent 开发：
    - Python / TypeScript SDK
    - 内置对话流程模板
    - 错误处理与重试机制
    - 测试工具与 Mock 服务
  </Accordion>

  <Accordion title="多服务网关" icon="diagram-project">
    Gateway 架构升级，支持多种本地生活服务：
    - 统一的服务抽象层
    - 可插拔的服务适配器
    - 跨服务的用户身份管理
  </Accordion>
</AccordionGroup>
