## Base URL

```
https://your-gateway/api/v1
```

## 认证

所有接口（除 Agent 注册外）需要 API Key 认证。大部分业务接口还需要 User Token。

```bash
# API Key 认证
Authorization: Bearer clw_your_api_key

# User Token 认证（业务接口需要）
X-User-Token: your_user_token
```

详见 [认证机制](/guides/authentication)。

## 统一错误格式

```json
{
  "error": {
    "code": "ERROR_CODE",
    "message": "人类可读的错误描述"
  }
}
```

详见 [错误处理](/guides/error-handling)。

## 限流

| 操作 | 限制 |
|------|------|
| 下单 (`POST /orders`) | 10 次/分钟/Agent |
| 其他接口 | 60 次/分钟/Agent |
| SMS 发送 | 60 秒冷却/手机号 |

超过限制返回 `429` 状态码和 `RATE_LIMITED` 错误码。
