## GET /orders/{order_id}

查询指定订单的状态和详情。

### 请求头

| 名称 | 必填 | 说明 |
|------|------|------|
| `Authorization` | 是 | `Bearer clw_...` |
| `X-User-Token` | 是 | 用户 Token |

### 路径参数

| 参数 | 类型 | 说明 |
|------|------|------|
| `order_id` | string | 外卖订单 ID（从创建订单返回）|

### 响应

```json
{
  "order_id": "1234567890",
  "status": "created",
  "shop_name": "瑞幸咖啡(国贸店)",
  "total_amount": 8.4,
  "created_at": "2026-03-23T10:15:30.123Z"
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `order_id` | string | 外卖订单 ID |
| `status` | string | 订单状态 |
| `shop_name` | string | 商家名称 |
| `total_amount` | float | 订单总额 |
| `created_at` | string | 创建时间（ISO 8601）|

### 示例

```bash
curl -X GET "https://your-gateway/api/v1/orders/1234567890" \
  -H "Authorization: Bearer clw_your_api_key" \
  -H "X-User-Token: your_user_token"
```
