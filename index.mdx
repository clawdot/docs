## 快速开始

<Tabs>
  <Tab title="REST API">
    ```bash
    # 搜索附近商家
    curl -X GET "https://gateway.clawdot.ai/api/v1/shops/search?lat=39.9&lng=116.4&keyword=咖啡" \
      -H "Authorization: Bearer clw_your_api_key" \
      -H "X-User-Token: your_user_token"
    ```
  </Tab>
  <Tab title="MCP Protocol">
    ```json
    {
      "tool": "search_shops",
      "arguments": {
        "lat": 39.9,
        "lng": 116.4,
        "keyword": "咖啡",
        "user_token": "your_user_token"
      }
    }
    ```
  </Tab>
</Tabs>
