## 双协议支持

<Tabs>
  <Tab title="REST API">
    标准 RESTful 接口，适合任何 HTTP 客户端：

    ```bash
    curl -X GET "https://your-gateway/api/v1/shops/search?lat=39.9&lng=116.4&keyword=咖啡" \
      -H "Authorization: Bearer clw_your_api_key" \
      -H "X-User-Token: your_user_token"
    ```
  </Tab>
  <Tab title="MCP Protocol">
    Model Context Protocol 接入，AI Agent 原生调用：

    ```json
    {
      "tool": "search_shops",
      "arguments": {
        "lat": 39.9,
        "lng": 116.4,
        "keyword": "咖啡",
        "user_token": "your_user_token"
      }
    }
    ```
  </Tab>
</Tabs>
